# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added

- Initial release of Drupal Coding Standards Agent Skill
- SKILL.md entry point with routing instructions
- standards-index.md file type routing table
- Composer VCS repository configuration for drupal/coding_standards
- README.md with installation and usage documentation
- Support for PHP, JavaScript, CSS, Twig, YAML, SQL, Markup, and Accessibility standards
- Dynamic context discovery optimization
- AGENTS.md integration documentation

### Notes

- Standards are automatically pulled from the official drupal/coding_standards repository
- Updates to standards can be obtained via `composer update drupal/coding_standards`
- This skill version tracks compatibility with the coding_standards repository structure
